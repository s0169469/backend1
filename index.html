  2. Использование команд /bin/bash, cd и ls

  3. С помощью команды ping на учебном сервере узнать IP-адрес веб-сервера kubsu.ru Ping - команда, которая используется для устранения неполадок подключения, доступности. Команда Ping отправляет на сервер пакеты маленького размера и считает время. На основании полученных данных можно сделать выводы о качестве соединения.

  4. С помощью команды nslookup узнать A-записи и MX-записи домена kubsu.ru и kubsu-dev.ru Команда nslookup позволяет узнать содержимое DNS : IP-адрес, типы записи. DNS - система доменных и имен, соединяет IP-адрес с доменным именем. Опция type=X указывает на тип записи, которую будем запрашивать. A-запись позволяет найти IP(с ее помощью домен прикрепляется к IP-адресу). Запись MX хранит соответствие доменного имени почтовому серверу.

  5. С помощью команды whois узнать дату регистрации домена kubsu.ru и kubsu-dev.ru whois представляет собой список записей, которые содержат в себе информацию о владельце домена
